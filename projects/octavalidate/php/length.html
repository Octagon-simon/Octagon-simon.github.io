LENGTH VALIDATION

Learn how you can check for input lengths with this library

This validation falls under the group Attributes Validation and they follow this common syntax.

//syntax
$valRules = array(
"FORM_INPUT_NAME" => array(
["ATTRIBUTE_TITLE", "VALUE", "CUSTOM_ERROR_MESSAGE"]
)
);

This Validation enables you to check the number of characters provided by the user.

LENGTH

Use this to check for exact number of characters. For Eg If you provide 5, it means that the user must provide 5 characters.

MINLENGTH

Use this to check for minimum number of characters. For Eg If you provide 5, it means that the user must provide 5 characters or more.

MAXLENGTH

Use this to check for maximum number of characters. For Eg If you provide 5, it means that the user must provide 5 characters or less.

Rule Title Possible value
LENGTH 5
MINLENGTH 10
MAXLENGTH 20

Follow the syntax below to define a length Validation for a form input name.

//syntax
$valRules = array(
"FORM_INPUT_NAME" => array(
["RULE_TITLE", "VALUE", "CUSTOM_ERROR_MESSAGE"]
)
);
//sample Validation
$valRules = array(
"username" => array(
["R", "Your username is required"],
["MINLENGTH", "5", "Your username must contain 5 characters or more"]
),
"age" => array(
["R", "Your age is required"],
["LENGTH", "2", "Your Age must contain 2 digits!"]
)
);
