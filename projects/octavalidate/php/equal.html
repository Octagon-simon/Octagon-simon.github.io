EQUALTO VALIDATION

This validation falls under the group Attributes Validation and they follow this common syntax.

Copy

//syntax
$valRules = array(
  "FORM_INPUT_NAME" => array(
      ["ATTRIBUTE_TITLE", "VALUE", "CUSTOM_ERROR_MESSAGE"]
 )
);

This validation checks if two input names have the same value. To use this validation, you need to define the EQUALTO rule on the first Input Name and then provide the second Input Name as a value to this validation rule.

Rule Title Possible value Input Name
EQUALTO confirm_password

This is useful if you need to validate a password and confirm password fields.

Follow the syntax below to use the EQUALTO validation rule.

Copy

//syntax
$valRules = array(
  "FIRST_INPUT_NAME" => array(
    ["EQUALTO", "SECOND_INPUT_NAME", "CUSTOM_ERROR_MESSAGE"]
  )
);
//sample validation
$valRules = array(
  "password" => array(
    ["R", "Your password is required"],
    ["EQUALTO", "confirm_password", "Both passwords do not match"]
  )
);
