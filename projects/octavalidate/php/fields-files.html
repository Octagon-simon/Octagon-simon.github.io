FIELDS & FILES VALIDATION

How to validate form fields and uploaded files

Assuming you have a form which allows users to submit data and to upload files, how would you validate the submitted data & uploaded files?

We know that the syntax to begin validation on the $_POST array or the $_GET array is;

//begin validation on $_POST array
$myForm->validateFields($valRules, $_POST);
//begin validation on $_GET array
$myForm->validateFields($valRules,$_GET);

And the method to begin validation on uploaded files is;

//begin validation on uploaded files
$myForm->validateFiles($valRules);

So what if you need to use both methods on your forms?

It's simple! Just create different variables and store the return value of the methods and then compare them.

//store validation of form fields
$fieldsVal = $myForm->validateFields($fieldRules, $_POST);
//store validation of uploaded files
$filesVal = $myForm->validateFiles($fileRules);
//compare and check if they were successful
if ($fieldsVal && filesVal){
//process form data here
}else{
//validation failed, print errors
print_r(json_encode($myForm->getErrors()));
