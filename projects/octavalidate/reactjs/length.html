LENGTH VALIDATION

This validation enables you to check the number of characters provided by the user.

LENGTH

Use this to check for exact number of characters. For Eg If you provide 5, it means that the user must provide 5 characters.

MINLENGTH

Use this to check for minimum number of characters. For Eg If you provide 5, it means that the user must provide 5 characters or more.

MAXLENGTH

Use this to check for maximum number of characters. For Eg If you provide 5, it means that the user must provide 5 characters or less.

Attribute Possible values Custom message
length 5 ov-length-msg
minlength 10 ov-minlength-msg
maxlength 20 ov-maxlength-msg
To prevent the browser from using its inbuilt maxlength & minlength validation on your form, add the attribute novalidate to the form tag.

DEMO FORM

<!-- length -->
<input id="inp_length" type="number" length="2" octavalidate="R">
<!-- min length-->
<input id="inp_min_length" type="text" minlength="5" octavalidate="R">
<!-- max length-->
<input id="inp_max_length" type="text" maxlength="10" octavalidate="R">
