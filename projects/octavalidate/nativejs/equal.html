EQUALTO VALIDATION

Learn how you can compare 2 input values using this library

This validation checks if two input fields have the same value. To use this validation, you need to provide the ID of the other input element to check against within the attribute equalto.

Attribute Possible values
Input element ID 		Custom message
equalto inp_password ov-equalto-msg

This is useful if you need to validate a password and confirm password fields.

DEMO FORM

<!-- password -->
<input id="inp_pass" type="password" octavalidate="R">
<!-- confirm password -->
<input id="inp_con_pass" type="password" equalto="inp_pass"  ov-equalto-msg="Both passwords do not match">
