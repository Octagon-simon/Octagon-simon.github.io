Range Validation

Learn how you can validate a range of numbers

This attribute validation helps to compare a range of numbers provided by the user. It is useful if you need to validate an input where a user must provide a number within a specified range.

Attribute Possible values Custom message
range 1 - 7 ov-range-msg

This type of validation only works on input type "number"

<input id="inp_range" type="number" range="1 - 5" ov-range-msg="Answer must be between 1-5">
