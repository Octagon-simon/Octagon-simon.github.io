OCTAVALIDATE FOR REACTJS

Client-side Validation Library for ReactJS

Octavalidate - ReactJS is a front-end validation library that you can import as a module and then use it within a Form component in React.

Before you continue, please note that the documentation on how to use this library is the same as the documentation for the Native JavaScript release.

INSTALL OCTAVALIDATE
$ npm install octavalidate-reactjs

Once the package has been installed;

  1. Import the module and
  2. Initialize the library on the page containing the form that you wish to validate.

login.jsx

import octaValidate from 'octavalidate-reactjs'

export default function Login() {
//handle form submission
const handleSubmit = function (e) {
//initialize the library
const myForm = new octaValidate('form_login')
//prevent reload
e.preventDefault();
//begin validation on form
if(myForm.validate()){
/** process form data here **/
}
}
//return the form
return(
<form id="form_login" method="post" noValidate onSubmit={handleSubmit}>
<input id="inp_email" name="email" type="email" octavalidate="R,EMAIL" />
<input id="inp_pass" name="password" type="password" octavalidate="R" minLength="8" />
<button type="submit">Login</button>
</form>
)
}

You can even choose to initialize the library with configuration options like the code below

import octaValidate from 'octavalidate-reactjs'

export default function Login() {
const configOpts = {
strictMode : true,
strictWords = ['admin', 'null', 'undefined']
}
//handle form submission
const handleSubmit = function (e) {
//initialize the library
const myForm = new octaValidate('form_login', configOpts)
}
}

Make sure the form ID exists and the input element ids are unique!
