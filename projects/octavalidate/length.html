LENGTH VALIDATION

Validate a form input that requires a number of characters from the user and show a custom message on validation failure using the attributes below.

Attribute Possible values Custom message
length 5 ov-length:msg
minlength 10 ov-minlength:msg
maxlength 20 ov-maxlength:msg
To prevent the browser from using its inbuilt maxlength & minlength validation on your form, add the attribute novalidate to the form tag.

DEMO FORM

<!-- length -->
<input id="inp_length" type="number" length="2" octavalidate="R">
<!-- min length-->
<input id="inp_min_length" type="text" minlength="5" octavalidate="R">
<!-- max length-->
<input id="inp_max_length" type="text" maxlength="10" octavalidate="R">
