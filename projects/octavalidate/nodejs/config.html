CONFIGURATION

The code below shows the syntax to initialise octavalidate for your form.

//import library
const octaValidate = require('octavalidate-nodejs')

//syntax
const validate = new octaValidate(FORM_ID, CONFIG_OPTIONS)

CONFIGURATION OPTIONS

To use any of the options below, provide it as an object and pass it as the second argument when creating an instance of octaValidate.

Option Description Possible value
strictMode Boolean This option removes extra white space from the start and at the end of a form input and also prevents the user from providing reserved keywords as values. Default value is false. true
strictWords Array This option alows you to provide words that users are not supposed to submit. For eg ["null", "error", "false"]. In order to use this option, you must set strictMode to true. ["null", "error", "false"]

To use any of the configuration options, I will create a new instance of the function like this:

//import library
const octaValidate = require('octavalidate-nodejs')
//config options
const config = {
strictMode : true,
strictWords : ["admin", "fake", "empty", "null"]
}
//create new instance
const validate = new octaValidate('my_form', config)
