EQUALTO VALIDATION

Learn how you can compare 2 input values using this library

This validation checks if two form fields have the same value. To use this validation, you need to provide the name of the other form field to check against. Here's the syntax

//syntax
const fieldRules = {
FIELD_NAME : {
EQUALTO : [OTHER_FIELD_NAME, ERROR_MESSAGE]
}
}

This is useful if you need to validate a password and confirm password fields.

Here's an example

//equalto validation
const fieldRules = {
password : {
'R' : "Your password is required",
'EQUALTO' : ['confirmPassword', 'Both passwords do not match']
}
confirmPassword : {
'R' : "Please re-enter your password"
}
}
