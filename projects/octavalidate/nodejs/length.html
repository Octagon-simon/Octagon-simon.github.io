LENGTH VALIDATION

Learn how you can check for input lengths with this library

This type of validation enables you to check the number of characters provided by the user. It is under the group Attributes Validation and all Attributes Validation follow the syntax below;

//syntax
const fieldRules = {
FIELD_NAME : {
RULE_TITLE : [VALUE, ERROR_MESSAGE]
}
}

LENGTH

Use this to check for exact number of characters. For Eg If you provide 5, it means that the user must provide 5 characters.

MINLENGTH

Use this to check for minimum number of characters. For Eg If you provide 5, it means that the user must provide 5 characters or more.

MAXLENGTH

Use this to check for maximum number of characters. For Eg If you provide 5, it means that the user must provide 5 characters or less.

Attribute Possible values
length 5
minlength 10
maxlength 20

Take a look at the rules below

//syntax
const fieldRules = {
age : {
"LENGTH" : [2, "Your age must be 2 digits"]
},
pass : {
"MINLENGTH" : [8, "Your password must have a minimum length of 8 characters"]
}
}
