CONFIGURATION

Assuming I have a form on my page, this is how I will create a new instance of the function.

<script>
//create new instance
const myForm = new octaValidate(FORM_ID, CONFIGURATION);
</script>

OPTIONS

To use any of the options below, provide it as an object and pass it as the second argument when creating an instance of octaValidate.

Option Description Possible values
strictMode Boolean This option removes extra white space from the start and at the end of a form input and also prevents the user from providing reserved keywords as values. Default value is false. true
successBorder Boolean This option removes extra white space from the start and at the end of a form input and also prevents the user from providing reserved keywords as values. Default value is false. true
strictWords Array This option alows you to provide words that users are not supposed to submit. For eg ["null", "error", "false"]. In order to use this option, you must set strictMode to true. ["null", "error", "false"]

To use any of the configuration options, I will create a new instance of the function like this: 

<script>
//configuration options
const myFormOptions = {successBorder: true, strictMode : true, strictWords : ["admin", "null", "undefined"]};
//create new instance of the function
const myForm = new octaValidate('my_form', myFormOptions);
</script>
