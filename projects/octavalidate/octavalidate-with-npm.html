INSTALLING WITH NPM

octaValidate is a front-end library, and before you can use it in your application, you have to install it through NPM an build the library with browserify.

Below are the steps to achieve this.

INSTALL BROWSERIFY

There are 2 ways to install browserify.

Install on your App

npm install browserify

Or Install globally

npm install -g browserify
INSTALL OCTAVALIDATE
npm install octavalidate

Now create a file ov.js that will store all form validations for your app.

Within this file, require the library, then initialize the function after the DOM is ready, and pass in the form ID and any configuration options as the second argument.

Learn more about configuration options here

ov.js

//require the module
const octaValidate = require ('octavalidate');
//initialise the function after DOM is ready!!!
document.addEventListener('DOMContentLoaded', function () {
  const myForm = new octaValidate('my_form');
  //listen for submit event
  document.querySelector('#my_form').addEventListener('submit', function(e){
    e.preventDefault();
    //invoke the method
    if(myForm.validate() === true){ 
      //validation successful, process form data here
    } else {
      //validation failed
    }
  })
})

Make sure to initialise the function after the DOM is ready!

BUILD WITH BROWSERIFY

There are 2 ways to build your validations with browserify.

Command-line

Use this method if you installed browserify globally.

Type the code below in a command-line to build your validations.

browserify ov.js -o validations.js

package.json

Open your app's package.json file and add / modify the lines below to build the ov.js.

"scripts": {
  "build" : "browserify ov.js -o validations.js"
}

Then run

npm run build

If you can see the file validations.js, it means that we have successfully built validations for the form. All you have to do now is to link validations.js to your HTML page and reload your server. 

<!-- form validation script -->
<script src="public/js/validations.js"></script>

<form id="my_form">
  <input octavalidate="R,EMAIL" id="inp_email" name="email" type="email">
  <input octavalidate="R,DIGITS" id="inp_age" name="age" type="number">
  <button type="submit">submit</button>
</form>

If you're using node js, make sure your app is set up to serve static files.
